# PlacePrep - Placement Preparation Platform

A comprehensive placement preparation platform for students to assess their skills, get personalized career guidance, and prepare for campus placements.

## Features

- **Skill Assessment**: Evaluate aptitude, coding, communication, and analytical skills
- **GATE & CAT Level Questions**: Prepare for competitive exams with challenging questions
- **Personalized Career Guidance**: Get job role recommendations based on your interests and performance
- **Company Insights**: Explore companies and their hiring requirements
- **Practice Resources**: Access learning resources and daily challenges

## Tech Stack

- React 18 with TypeScript
- Vite for fast development
- Tailwind CSS for styling
- shadcn/ui for UI components

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```
